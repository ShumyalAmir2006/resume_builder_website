# Resume Builder Website

A simple resume builder web application that allows users to create and preview resumes directly in the browser.

## Features

- Live resume preview
- Add and remove experience, education, and projects
- Skills section with tags
- Auto save using browser LocalStorage
- Export resume as PDF
- Backup and restore resume data using JSON
- Fully frontend based (no backend required)

## Tech Stack

- HTML
- CSS
- JavaScript
